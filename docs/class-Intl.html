Class Intl

Intl like Gettext

This class can translate application texts from arrays or JSON or Gettext PO files with plurals.

It can auto-detect the user preferred languages and load translation files in JSON format or PHP scripts with arrays of strings can to work with Gettext PO files with plurals. It can also format strings with placeholders %named_value to translated string 'named_value'.
